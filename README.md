![image](https://user-images.githubusercontent.com/14907987/227293599-a628f81c-b867-4495-a692-cc7c620365df.png)

## Description
A THREE.JS virtual photo gallery. A 3D exhibition to show some of my landscapes prints.

## Install
```sh
npm install
```

## Build
```sh
npm run build
```

## Serve
```sh
npx serve -s public
```
